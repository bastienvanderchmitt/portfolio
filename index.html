Réalisations

Stage INRA

J'ai réalisé mon stage de fin d'études de l'IUT à l'INRA (Institut National de Recherche Agronomique). J'ai conçu un site web qui permet de contrôler l'acquisition de données d'une machine construite par les chercheurs qui mesure comment "respire" un fruit ou un légume. Pour ce faire, j'ai utilisé le framework WebDev et le protocole FTP afin de connecter les deux entités. Rapport de Stage

Framework Applis Mobiles

Aujourd'hui, le marché des smartphones est en train d'exploser. Or, la diversité des systèmes d'exploitation pose des problèmes aux développeurs d'applications mobiles. C'est pourquoi, dans le cadre de la Faculté des Sciences de Montpellier, j'ai réalisé, à l'aide de deux collègues, une Ligne de produits qui permet d'avoir un seul code source pour une même application dans des plateformes différentes : Framework Rizzla
